# 多能力一夜人狼ゲーム（改良版）

このリポジトリは「多能力一夜人狼」ゲームのUI改善版です。以下の改良を行っています：

- 画面レイアウトの最適化（PCで1画面表示）
- 山札から役職カードが配られるアニメーション追加
- 役職カードのデザイン強化と画像の活用

1. UI/UXの改善
